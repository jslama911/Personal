Chart Maker

Respiratory Equations

I currently work in the medical field as a Respiratory Therapist and I wanted to make something that could use that knowledge.

This is aa application that allows the user to choose from a list of medical equations.

It then allows the user to input the values needed to calculate the equation.

If the equation that someone is not listed there I also included the above calculator to allow the user to do any other calculations they could need.
